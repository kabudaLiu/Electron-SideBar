#### 可以实施页面的显示隐藏、桌面画画、橡皮擦以及撤销等简单的功能

## Installation✍
```python 
cd Election-SideBar
npm install
cd SideBar
npm install
```

##    Quick start✌
```python 
cd SideBar
npm run dev
cd ..
npm run dev
npm run start
```
